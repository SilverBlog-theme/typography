## 安装

```bash
cd templates
curl https://raw.githubusercontent.com/SilverBlogTheme/typography/master/install.sh | bash
```

## 定制

### 改变配色方案

主题配置文件的 `Dark_Theme` 字段将控制配色方案。

如果为真，则使用暗色主题，否则使用浅色主题。
